A simple path library written in Swift.

## Installation

Add the following dependency to your `Package.swift`:
```swift
.package(url: "https://github.com/sersoft-gmbh/pathwrangler.git", from: "1.0.0"),
```

Or add it via Xcode (as of Xcode 11).
